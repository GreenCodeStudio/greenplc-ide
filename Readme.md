# GreenPLC IDE
GreenPLC IDE is a free (MIT License) graphical IDE for programming microcontrollers using block/diagram-based editors. It focuses on rapid prototyping and generating C code targeting for example Arduino and AVR platforms.

## Features
- Drag-and-drop visual programming interface
- Generates C code for Arduino
- Generates machine code for Atmel AVR (work in progress)
- Live deploy to Arduino boards (work in progress)

## How to start
* Open the [GreenPLC IDE](https://greencodestudio.github.io/greenplc-ide/) in your browser.
* Select directory on your computer where project files will be stored.
* Create a new .gplc file.
* Create device module for your board (e.g. Arduino Nano)
