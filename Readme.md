# GreenPLC IDE
It's a free (MIT License) IDE for programming microcontrollers using graphical diagrams.

## Features
* Drag and drop interface for creating programs
* Generates C code for Arduino
* Generates machine code for Atmel AVR (for example ATmega microcontrollers used in Arduino boards) (in progress)
* Deploys programs to Arduino boards without restarting the board (in progress)
